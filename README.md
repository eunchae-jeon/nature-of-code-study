# Nature of Code Study

This project is a study of the concepts presented in "The Nature of Code" by Daniel Shiffman, utilizing p5.js for interactive examples. The project is structured into chapters, each containing relevant examples and explanations.

## Project Structure

```
nature-of-code-study
├── src
│   ├── chapters
│   │   ├── chapter1.html
│   │   ├── chapter2.html
│   │   └── chapter3.html
│   ├── examples
│   │   ├── chapter1
│   │   │   └── example1.js
│   │   ├── chapter2
│   │   │   └── example1.js
│   │   └── chapter3
│   │       └── example1.js
│   ├── styles
│   │   └── main.css
│   ├── scripts
│   │   └── menu.js
│   └── index.html
└── README.md
```

## Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd nature-of-code-study
   ```

3. **Open the `index.html` file** in your web browser to view the project.

## Chapter Overview

- **Chapter 1**: Introduction to p5.js and basic concepts.
- **Chapter 2**: Intermediate concepts and examples.
- **Chapter 3**: Advanced topics and applications.
